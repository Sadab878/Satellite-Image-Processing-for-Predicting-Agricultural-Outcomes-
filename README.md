# Satellite-Image-Processing-for-Predicting-Agricultural-Outcomes-
Tech Stack :- Python, Machine Learning, Remote Sensing
Analyzed Indian Pines hyperspectral data to detect subtle variations in crop health, chlorophyll, and nutrient levels. 
Utilized ENVI software to process high-dimensional spectral bands for precision agricultural monitoring. 
Implemented spectral indices (NDVI) to evaluate vegetation stress and optimize resource allocation.
